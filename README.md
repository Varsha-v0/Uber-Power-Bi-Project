# Uber-Power-Bi-Project

* Analyze Uber trip data to identify travel patterns and ride demand

* Understand peak hours and busiest days

* Evaluate trip distance, duration, and fare trends

* Visualize key performance indicators (KPIs) using interactive dashboards.

* Provide insights that could help optimize ride allocation and driver availability.
